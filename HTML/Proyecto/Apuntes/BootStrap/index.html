Introduccion

Bootstrap es un framework CSS y Javascript diseñado para la creación de interfaces limpias y con un diseño responsive. Además, ofrece un amplio abanico de herramientas y funciones, de manera que los usuarios pueden crear prácticamente cualquier tipo de sitio web haciendo uso de los mismos. Esta mima página esta creada con bootstrap+css por ejemplo.

Puesta a punto de Bootstrap

Para poder utlizar bootstrap tenemos que añadir las librerías a nuestro archivo html.

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js"></script>

<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js"></script>

Posicionamiento en Bootstrap

Bootstrap utiliza un sistema de rejillas que permite usar hasta 12 columnas en una pagina, puedes definir columnas del tamaño que elijamos.

Para posicionamiento en rejilla tanto columnas como filas, indicando en la class del section/article/div que queramos la clase "col" seguido del de un guión y el tamaño que queramos darle (de 1 a 12). Después en un sub div/section/article con clase de tipo row para indicarle la o las filas que queremos en dicha columna

Responsive

Para el metodo resposive tenemos 5 clases para indicarle el tamaño de pantalla

Elemento Descripcion
(Por defecto, no hace falta poner la clase) Dispositivos extra pequeños - ancho de pantalla inferior a 576 px
-sm Dispositivos pequeños: ancho de pantalla mayor o igual a 576 px
-md Dispositivos medianos: ancho de pantalla mayor o igual a 720 px
-lg Dispositivos largos: ancho de pantalla mayor o igual a 960 px
-xl Dispositivos xlarge: ancho de pantalla mayor o igual a 1140px.

Por ejemplo: <div class="col-md-6 col-sm-8"> Este div, en pantallas medianas ocupará 6/12 y en dispositivos pequeños ocupará 8/12 columnas disponibles.

Colores

En Bootstrap tenemos colores para añadir tanto a texto, fondo, lo que queramos.

Elemento Descripcion
-primary Color Azul oscuro
-secondary Color Gris oscuro
-success Color Verde
-danger Color Rojo
-warning Color Amarillo
-info Color Azul claro
-light Color Blanquecino
-info Color Negro

Por ejemplo: <div class="col-md-6 col-sm-8"> Este div, en pantallas medianas ocupará 6/12 y en dispositivos pequeños ocupará 8/12 columnas disponibles.
