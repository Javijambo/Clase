Introducción

Materialize es un framework basado en Material Design, un concepto creado por Google para unificar el diseño de sus aplicaciones sin importar la plataforma en que se muestren.

Puesta a punto de Materialize

Para poder utlizar Materialize tenemos que añadir las librerías a nuestro archivo html.

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.97.3/css/materialize.min.css">

<script type="text/javascript" src="https://code.jquery.com/jquery-2.1.1.min.js"> </script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.97.3/js/materialize.min.js"> </script>

Posicionamiento en Materialize

Materialize utiliza un sistema de rejillas que permite usar hasta 12 columnas en una pagina sin importar el navegador, cada una de ellas tendra el mismo ancho siempre.

Para posicionamiento en rejilla tanto columnas como filas, indicando en la class del section/article/div que queramos la clase col n donde n es el tamaño que queramos darle a la columna (de 1 a 12). Después en un sub div/section/article con clase de tipo row para indicarle la/s fila/s que queremos en dicha columna

Si queremos posicionar una columna a la derecha podemos poner en la clase offset sn donde n es el numero de columnas que queremos restantes a la derecha.

Helpers

En Materialize tenemos clases para alinear contenidos.

Elemento Descripcion
valign-wrapper Alinea el contenido verticalmente
left-align Alinea el TEXTO a la izquierda
right-align Alinea el TEXTO a la derecha
center-align Alinea el TEXTO a el centro
left Hace un float a la izquierda
right Hace un float a la derecha

Responsive

Para el metodo resposive tenemos 5 clases para indicarle el tamaño de pantalla

Elemento Descripcion Width del contenedor
s Dispositivos extra pequeños - ancho de pantalla inferior a 576 px 90%
m Dispositivos pequeños: ancho de pantalla mayor o igual a 576 px 85%
l Dispositivos medianos: ancho de pantalla mayor o igual a 720 px 70%
xl 70% Dispositivos largos: ancho de pantalla mayor o igual a 960 px

Por ejemplo: <div class="col s12 m4 l3"> Este div, en pantallas grandes ocupará 3 columnas, 4 en medianas y 12 en pequeñas.
