Introducción

Introducción CSS Grid es un modelo de maquetación CSS en base a una rejilla con filas y columnas, lo que facilita el diseño de páginas web sin tener que usar floats y posicionamiento.

Posicionamiento

El sistema de grid se basa en un elemento padre con uno o varios elementos hijo, utiliza un sistema de columnas para posicionar verticalmente los elementos y filas para horizontalmente

Columnas

Filas

Con la propieda grid-column-start y grid-column-end indicamos donde empieza y donde acaba el elemento en columnas

Alineamiento

Para ello tenemos las clases Justify content y Align-content

Justify-content se utiliza para alinear horizontalmente toda la cuadrícula dentro del contenedor

Align-content se utiliza para alinear verticalmente toda la cuadrícula dentro del contenedor

Tenemos distintas propiedas para utilizarlos:

Elemento Descripción
start Los elementos se alinean al inicio del container.
end Los elementos se alinean al inicio del container.
center Los elementos estan alineados en el centro.
space-between Los elementos se alinean equitativamente con el primer elemento y el ultimo.
space-around Los elementos están distribuidos de forma que tengan el mismo espaciado entre ellos.
space-evenly Los elementos se alinean de manera que el espacio de detras al elemento anterior y el espacio de delante del elemento siguiente sea el mismo.

Responsive

Para el metodo resposive tenemos 5 clases para indicarle el tamaño de pantalla

Elemento Descripcion Width del contenedor
s Dispositivos extra pequeños - ancho de pantalla inferior a 576 px 90%
m Dispositivos pequeños: ancho de pantalla mayor o igual a 576 px 85%
l Dispositivos medianos: ancho de pantalla mayor o igual a 720 px 70%
xl 70% Dispositivos largos: ancho de pantalla mayor o igual a 960 px

Por ejemplo: <div class="col s12 m4 l3"> Este div, en pantallas grandes ocupará 3 columnas, 4 en medianas y 12 en pequeñas.
