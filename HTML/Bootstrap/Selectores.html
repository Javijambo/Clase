  • Universal(*): Sirve para seleccionar todos los elementos
  • De etiqueta(X): Sirve para seleccionar todos los elementos de esa etiqueta, se pueden concatenar etiquetas poniendo una coma entre ellas
  • Descendente (X X): Sirve para elegir etiquetas o elementos dentro de otras etiquetas.
  • Selector de clase (.X): Sirve para elegir una etiqueta o elemento en particular añadiendole en la etiqueta class="nombreclase"
  • Selectores de id (#X): Exactamente igual que antes pero con la clase id="nombreid"
  • Selector de hijos (X>X): Similar al descendente pero se utiliza para seleccionar un elemento que es hijo directo de otro elemento
  • Selector adyacente (X+X): El selector adyacente se emplea para seleccionar elementos que en el código HTML de la página se encuentran justo a continuación de otros elementos
  • Los cuatro tipos de selectores de atributos son:
    • [nombre_atributo], selecciona los elementos que tienen establecido el atributo llamado nombre_atributo, independientemente de su valor.
    • [nombre_atributo=valor], selecciona los elementos que tienen establecido un atributo llamado nombre_atributo con un valor igual a valor.
    • [nombre_atributo~=valor], selecciona los elementos que tienen establecido un atributo llamado nombre_atributo y al menos uno de los valores del atributo es valor.
    • [nombre_atributo|=valor], selecciona los elementos que tienen establecido un atributo llamado nombre_atributo y cuyo valor es una serie de palabras separadas con guiones, pero que comienza con valor. Este tipo de selector sólo es útil para los atributos de tipo lang que indican el idioma del contenido del elemento.
  • Pseudoclases (X:pseudoclase): son una palabra clave añadida a los selectores, que especifica un estado especial del elemento a seleccionar.
    • :link -Un enlace que no se ha visitado
    • :visited -Un enlace que se ha visitado
    • :active -Un elemento (como un botón) que el usuario está activando.
    • :hover -Cuando "se pasa por encima" del elemento
    • :focus -Cuando el usuario hace clic, toca un elemento o lo selecciona con la tecla "Tab" del teclado.
    • :first-child -El primer elemento de un grupo de elementos.
    • :last-child -El ultimo elemento de un grupo de elementos.
    • :nth-child(n) -Uno o más elementos en función de su posición entre un grupo de hermanos.
    • :nth-last-child(n) -Selecciona uno o más elementos en función de su posición entre un grupo de hermanos, contando desde el final.
    • :nth-of-type(type) -Selecciona uno o más elementos de un tipo dado, en función de su posición entre un grupo de hermanos.
    • :first-of-type(type) -Representa el primer elemento de su tipo entre un grupo de elementos hermanos.
    • :last-of-type(type) -Representa el ultimo elemento de su tipo entre un grupo de elementos hermanos.
    • :empty Representa cualquier elemento que no tenga hijos
    • :target -Representa un elemento único (el elemento objetivo) con un id que coincide con el fragmento de la URL.
    • :checked -Representa cualquier radio,checkbox u option que este seleccionado
    • :enabled -Cualquier elemento habilitado.
    • :disabled -Cualquier elemento deshabilitado.
